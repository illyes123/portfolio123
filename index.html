Projet principal

Préparation du parc pour le déploiement de nouvelles machines

🎯 Objectif de la mission

Préparer le parc informatique pour recevoir de nouvelles machines en migrant les postes existants vers Windows 11. Utilisation de WAPT pour le déploiement automatisé, avec solution de secours via clé USB bootable.

📅 Déroulement du projet

Étape 1 - Analyse

Identification des postes à migrer

Utilisation de GLPI et OCS pour lister les machines encore sous Windows 10 ou versions antérieures.

Étape 2 - Localisation

Repérage physique avec Télémétrobox NG

Identification du port switch et de l'emplacement exact de chaque poste dans les locaux.

Étape 3 - Déploiement

Migration via WAPT ou clé USB

Descente de l'image Windows 11 sur les postes identifiés, avec vérification post-installation.

Étape 4 - Validation

Tests et réintégration au domaine

Vérification du bon fonctionnement et jonction au domaine Active Directory.
